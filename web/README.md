## Getting Started

1. Clone the repository

```bash
git clone https://github.com/jupiverse/jupiverse-kit.git
```

2. Navigate to the web directory

```bash
cd web
```

3. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```
